CT A Embed — Parent Integration

This example shows how to embed the CTA and listen for events. Allowed parent origins include tharaga.co.in and auth.tharaga.co.in.

Embed

Listen for events

window.addEventListener('message', (ev) => {
  if (!/tharaga\.co\.in|localhost|127\.0\.0\.1/.test(new URL(document.referrer || location.href).origin)) return;
  if (ev.data && ev.data.type === 'tharaga_lead_ok') {
    console.log('Lead OK', ev.data.payload);
  }
  if (ev.data && ev.data.type === 'tharaga_lead_error') {
    console.warn('Lead error', ev.data.error);
  }
});
